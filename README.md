# Galeria de Artes Responsiva

A semantic web page project developed to showcase a gallery of art using modern CSS techniques.



• Core Technical Focus

Semantic HTML: Built with a clean hierarchy using <header>, <main>, <article>, and <footer> tags.

Flexbox Layout: Uses a dynamic .art-container with display: flex and flex-wrap for organized content.

Responsive Design: Includes Media Queries for mobile and tablet views using relative units like rem, vw, and vh.

CSS Variables: Utilizes :root variables for consistent colors and fonts across the project.



• Key Features

Fixed Navigation: A top navigation bar that stays in place while scrolling.

Art Portfolio: Interactive display of artworks with titles and descriptions.

Feedback System: A dedicated comments section with a functional <form> for user input.



• How to Use

Open index.html in any browser.

Navigate through sections using the menu links (#artes, #exposicoes, #comentarios)
